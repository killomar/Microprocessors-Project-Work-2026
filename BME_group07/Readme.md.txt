Project Title: Bedside Patient Emergency Call System Using Arduino

Description:
This project simulates a patient emergency alert system using Arduino in Tinkercad.

Components:
- Arduino Uno
- Push Button
- LCD Display
- Buzzer
- LED

Features:
- Button press triggers alert
- LCD displays emergency message
- LED and buzzer activate
